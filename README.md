# Sistema de Gerenciamento de ONGs

## Sobre o Projeto

Este é um projeto acadêmico desenvolvido para demonstrar o desenvolvimento completo de um **Sistema de Gerenciamento para Organizações Não Governamentais (ONGs)**, desde a estruturação básica até a implementação de funcionalidades avançadas com foco em acessibilidade e otimização para produção.

## Contexto Acadêmico

O projeto foi desenvolvido em 4 etapas práticas progressivas, cada uma adicionando novas funcionalidades e melhorias à aplicação:

### Estrutura das Experiências Práticas

#### **Experiência Prática 1 - Estrutura HTML5**
- Estruturação semântica do projeto
- Criação das páginas principais (Início, Projetos, Cadastro)
- Implementação de formulários básicos
- Organização de arquivos do projeto

#### **Experiência Prática 2 - Design e Estilização**
- Design visual responsivo com CSS3
- Sistema de cores institucional
- Componentes estilizados (navegação, botões, formulários, cards)
- Responsividade para mobile, tablet e desktop
- Layout adaptativo com breakpoints estratégicos

#### **Experiência Prática 3 - Funcionalidades Interativas**
- Desenvolvimento com JavaScript
- Navegação dinâmica e responsiva
- Formulários inteligentes com validação
- Máscaras de entrada (CPF, telefone, CEP)
- Interatividade completa do menu e componentes

#### **Experiência Prática 4 - Acessibilidade e Otimização**
- Implementação de acessibilidade WCAG 2.1 Nível AA
- Navegação por teclado completa
- Suporte a leitores de tela
- Modo alto contraste e modo escuro
- Minificação e compressão para produção
- Otimização de imagens e arquivos

## Como Começar

### Clone o Repositório
```bash
git clone https://github.com/Katarinaberninii/administracao-ong.git
cd administracao-ong
```

### Acesse uma Experiência Prática
```bash
# Experiência 1
cd experiencia-pratica-1

# Ou Experiência 2
cd experiencia-pratica-2

# Ou Experiência 3
cd experiencia-pratica-3

# Ou Experiência 4 (Mais completa)
cd experiencia-pratica-4
```

### Instale as Dependências (Experiência 4)
```bash
npm install
```

### Inicie o Servidor
```bash
# Opção 1: Usar Python
python -m http.server 8000

# Opção 2: Usar npm (Experiência 4)
npm start
```

Abra seu navegador em `http://localhost:8000`

## Recursos Principais

### Funcionalidades
- Páginas informativas sobre a ONG
- Formulários de cadastro e registro
- Design responsivo e acessível
- Navegação completa por teclado
- Modo escuro e alto contraste
- Totalmente responsivo

### Acessibilidade (WCAG 2.1 AA)
- Navegação por teclado
- Estrutura semântica apropriada
- Contraste mínimo de 4.5:1
- Suporte a leitores de tela
- Modo de alto contraste
- Tamanho de texto ajustável

### Otimização para Produção
- Minificação de HTML, CSS e JavaScript
- Compressão de imagens
- Geração de WebP
- Build automatizado com Grunt
